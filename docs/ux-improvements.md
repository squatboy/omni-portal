# UX 개선 사항

## 스크롤바 레이아웃 흔들림(Layout Shift) 방지

### 문제 상황
페이지 콘텐츠의 길이에 따라 스크롤바가 생기거나 없어질 때, 화면 내 컴포넌트들이 가로로 미세하게 움직이는 레이아웃 흔들림 현상 발생.

### 해결 방법
글로벌 스타일(`frontend/app/globals.css`)의 `html` 선택자에 `scrollbar-gutter: stable` 속성을 추가함.
이를 통해 스크롤바가 존재하지 않는 상태에서도 스크롤바 영역을 미리 확보하여, 페이지 길이에 관계없이 컴포넌트들의 레이아웃 위치가 항상 일정하게 유지되도록 개선함.
